# 示范领导

> “这个你先看一下，明天我们再对齐一下。”

如果你只是想先看效果，不想先跑命令，那就从这个目录开始。

这个示例模拟的是一种典型的：

- 口头优先级偏高
- 会前补需求
- 默认先给初稿
- 同步时强调口径和节奏

## 先看这几个文件

1. [persona.md](./persona.md)
2. [intent-map.md](./intent-map.md)
3. [playbook.md](./playbook.md)
4. [versions/20260401-initial/](./versions/20260401-initial)

## 这个示例能告诉你什么

- 领导常说的话，真实意思可能是什么
- 哪些话是高优先级信号，哪些话是背锅预警
- 你应该怎么回，怎么问，怎么在同步时保护自己

## 为什么它不是纯梗

因为它不是只生成一段吐槽文案，而是生成一个完整 bundle：

- 画像
- 意图地图
- 生存手册
- 版本快照

也就是说，你不只是在“玩一下”，而是在积累一个可修正、可进化的领导画像。
