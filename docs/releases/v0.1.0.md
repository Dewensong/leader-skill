# v0.1.0

## What changed

- launched `leader-skill` as a local-first boss-language decoder
- added CLI commands for create, translate, priority, reply, risk, promotion, rollback, and delete flows
- added local source routing for text, screenshots, Markdown, PDF, chat exports, and email exports
- added versioned leader profile generation with persona, intent map, and survival playbook outputs
- added bilingual documentation, install guide, release checklist, assets, and GitHub templates

## Why it matters

This release turns the recent “meme skill” wave into a more useful workplace tool:

- not a reskin
- not personality cosplay
- instead: decode vague boss messages into actual requirements, hidden risks, and safer replies

## Validation

- `python -m unittest discover -s tests -p "test_*.py"`
- `python -m tools.cli translate --text "这个你先看一下，晚点我们再对齐。"`
- `python -m tools.cli create-leader --slug demo-boss --name "示例领导" --text "这个不复杂吧，你今天先出个方案，明天我们同步一下。"`
