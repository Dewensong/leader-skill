# v0.1.1

## What changed

- switched human-facing CLI commands to Markdown-first output by default
- added richer persona summaries and follow-up questions in analysis results
- upgraded generated leader bundles with a quick-glance `README.md`
- refreshed the sample bundle under `examples/demo-leader/`
- rewrote README, INSTALL, and architecture docs to present the project more like a polished skill product page
- updated the terminal demo SVG to match the current Markdown output

## Why it matters

The repository now feels closer to the “high-conversion” style of the reference skill projects:

- easier to understand in one glance
- easier to trust without running code first
- easier to demo in screenshots or social posts
